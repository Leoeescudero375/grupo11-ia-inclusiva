Pixel — El asistente del sitio

Pixel es el asistente conversacional de este proyecto. Fue creado desde cero en Landbot, siguiendo los principios de diseño de chatbots educativos desarrollados durante la cursada: objetivos claros, flujo conversacional definido, tono cálido y pausado, sin recolección de datos personales.

Pixel responde preguntas frecuentes sobre el proyecto a través de un menú de opciones claras. El visitante elige qué quiere saber y recibe una respuesta breve, amigable y directa.

Cómo crear Pixel en Landbot

El chatbot fue diseñado desde cero en landbot.io. A continuación el flujo para reproducirlo:

  1. Crear cuenta gratuita en Landbot → Web Chatbot.
  2. Bloque de bienvenida: mensaje inicial de Pixel.
  3. Bloque de botones: las 5 opciones del menú.
  4. 5 bloques de respuesta, uno por opción.
  5. Cada respuesta conecta de vuelta al menú principal.
  6. Personalizar colores (azul #4A7FA5) y avatar.
  7. Publicar y copiar el snippet <script> de embed.
  8. Pegar el snippet antes de </body> en cada página del sitio.

Nota: Pixel no recopila datos personales y no reemplaza la mediación docente. Complementa el recorrido del visitante por el sitio.

Integración en el sitio

Pixel aparece en todas las páginas

Una vez generado el código embed de Landbot, se inserta en cada archivo HTML antes del cierre de </body>. El botón flotante naranja visible en la esquina inferior derecha de este sitio enlaza a esta página como referencia hasta que el chatbot esté publicado. Cuando Pixel esté listo en Landbot, reemplazá ese enlace por el snippet de embed y el chatbot aparecerá como burbuja flotante en todo el sitio.

Fragmento de código de integración

<!-- LANDBOT EMBED — reemplazar con tu snippet de Landbot --> <script src="https://cdn.landbot.io/landbot-3/landbot-3.0.0.js"></script> <script> var myLandbot = new Landbot.Popup({ configUrl: 'https://storage.googleapis.com/landbot.online/v3/TU-BOT-ID/index.json', }); </script>

El valor TU-BOT-ID se obtiene desde el panel de Landbot al publicar el bot → Share → Embed.
